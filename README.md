# Life Expectancy Analysis

## Overview

Life Expectancy Analysis is a machine learning-based project that predicts life expectancy using various health and socio-economic factors. The model is trained using a dataset and integrated with a Flask web application to allow users to input data and obtain predictions.

---

## Technologies Used

* **Python**
* **Flask**
* **Scikit-learn (sklearn)**
* **Pandas**
* **NumPy**

---

## Key Features

* Data preprocessing and analysis
* Machine learning model for prediction
* Web interface built using Flask
* User input for generating predictions

---

## How the Project Works

1. The dataset is loaded and processed using **Pandas**.
2. Data cleaning and feature selection are performed.
3. A machine learning model is trained using **Scikit-learn**.
4. The trained model is integrated into a **Flask web application**.
5. Users can input data through the interface and receive life expectancy predictions.
